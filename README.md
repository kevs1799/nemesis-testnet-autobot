# 🚀 Nesmesis Trade Testnet Autobot
---

## ⚙️ Requirements

 install:

- Node.js (v16 / v18 recommended)
- npm 
- Git

---

## 📥 Clone Repository

```bash
git clone https://github.com/kevs1799/nemesis-testnet-autobot.git
cd nemesis-testnet-autobot
```

## Install Dependencies

```bash
npm install
```

## Privatekey Import
edit pk.txt, peplace with yours


## Run

```bash
node index.js
```
note : this bot will run everyday
